# Future Updates

Ideas we like but are not building yet:

- Retry these misses: after a round, offer a quick retry round using only the words missed in that round.
- Storage used indicator: show an approximate local storage size in the Dashboard.
- Optional audio later: pronunciation is useful, but should be handled carefully so the app stays light.
- Story packs: add more relatable stories for each level and topic.
- Placement check: if someone is not sure of their level, offer a short friendly placement quiz.
- Goal history: show weekly goal streaks without turning it into a noisy streak system.
